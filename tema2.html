TEMA 2

Grundlæggende Web

I dette tema blev jeg introduceret til HTML og CSS, som jeg brugte til at omsætte wireframes til responsive løsninger. Jeg arbejdede med grid og flex til layout og anvendte designprincipper som Gestaltlovene, visuelt hierarki og farvelære for at skabe brugervenlige og æstetiske designs. Formålet var at kombinere tekniske færdigheder med designforståelse og formidle processen klart til interessenter.

Skabe knapper med CSS og HTML

I dette forløb lærte jeg at lave knapper ved at kombinere hypertext anchors og CSS-styling på en div. Jeg brugte CSS-egenskaber som padding, baggrundsfarve og hover-effekter for at skabe knapper, der både var funktionelle og æstetisk tiltalende.

Placering af elementer

Jeg lærte at anvende flex-column sammen med last-child-selektoren for præcist at kunne placere elementer, som f.eks. billeder, i bunden af en container. Denne metode gav mig bedre kontrol over layoutets struktur og design.

Fleksible gallerier med CSS Grid

Jeg lærte at bruge CSS Grid til at opbygge fleksible gallerier, hvor billeder kan spænde over flere rækker og kolonner. Dette gav mig mulighed for at skabe dynamiske og visuelt spændende layouts, der tilpasses forskellige skærmstørrelser.
